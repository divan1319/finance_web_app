# Aplicación web para el control de finanzas

## Configuración

### Configurar las variables para la bd

```bash
DB_CONNECTION=mysql
DB_HOST=mysql
DB_PORT=
DB_DATABASE=
DB_USERNAME=
DB_PASSWORD=

FORWARD_DB_PORT=
```

### Inicializar el proyecto:

```bash
 chmod +x init.sh
./init.sh
```



### Ejecutar el proyecto

```bash
./vendor/bin/sail up -d
```
